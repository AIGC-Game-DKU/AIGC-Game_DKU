Project Overview

Interactive Narrative Innovation

Our project focuses on developing a new type of interactive narrative game that leverages AI to create dynamic, evolving stories. Unlike traditional games that rely on fixed, pre-written scripts, our game allows the story to change based on the player's actions.

Key Features

  • Dynamic NPC Interactions
  • Adaptive Storytelling
  • Real-time World Evolution
